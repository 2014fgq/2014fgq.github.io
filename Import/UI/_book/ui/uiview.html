UIView

UIView

  • 什么是控件？

    • 屏幕上的所有UI元素都叫做控件，也有人叫做视图、组件 按钮（UIButton）、文本（UILabel）都是控件
    • 所有的控件最终都继承自UIView

  • 控件的共同属性有哪些？

    • 尺寸
    • 位置
    • 背景色
    • ......

父子控件（自定义控件基础）

  • 每个控件都是个容器，能容纳其他控件

    • 内部小控件是大控件的子控件
    • 大控件是内部小控件的父控件

  • 父子控件属性及方法

    • 常用属性

      • superview：获得自己的父控件对象

        • 每个子控件的superview只有一个，只读
          @property(nonatomic,readonly) UIView *superview;

      • subviews：获得自己的所有子控件对象

        • subviews是一个数组，只读
          @property(nonatomic,readonly,copy) NSArray *subviews;

      • tag：控件的ID(标识)，父控件可以通过tag来找到对应的子控件

        • tag值是随意设置，很容易混乱，能不用就不用
          @property(nonatomic) NSInteger tag;

    • 常用方法

      • addSubview：添加一个子控件view

        -(void)addSubview:(UIView *)view;

      • removeFromSuperview：从父控件中移除

        -(void)removeFromSuperview;

      • viewWithTag：根据一个tag标识找出对应的控件（一般都是子控件）

        -(UIView *)viewWithTag:(NSInteger)tag;

UIView位置、尺寸属性

  • frame：控件矩形框在父控件中的位置和尺寸(以父控件的左上角为坐标原点)

    @property(nonatomic) CGRect frame;

  • bounds：控件矩形框的位置和尺寸(以自己左上角为坐标原点，所以bounds的x、y一般为0)

    • 调整bounds的x,y值没有效果。设置bounds伸缩是以中心点为基准伸缩的
      @property(nonatomic) CGRect bounds;

  • center：控件中点的位置(以父控件的左上角为坐标原点)

    @property(nonatomic) CGPoint center;

  • transform：控件的形变属性(可以设置旋转角度、比例缩放、平移等属性)

    • 功能强大
      @property(nonatomic) CGAffineTransform transform;
